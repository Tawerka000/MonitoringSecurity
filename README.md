# MonitoringSecurity

Датасеты находятся в папке Datasets\NewDataSets. Для каждого типа атаки отдельный датасет.

В папке TeachingNeuralNetwork лежат скрипты для обучения нейронной сети на основе датасетов.

Обученные модели сохраняются в папку NeuralNetwork_models_saves. Оттуда же они будут подгружаться при использовании в основном приложении.

Файлом запускающим всё приложение является AppInterface.py

При запуске на другой системе необходимо будет изменить пути к датасетам и для сохранения/использования моделей.
